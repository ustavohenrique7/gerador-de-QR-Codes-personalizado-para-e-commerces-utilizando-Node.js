📱 Gerador de QR Code para E-commerce (Node.js)

Este projeto é um gerador de QR Codes desenvolvido em Node.js que roda diretamente no terminal. Ele permite criar QR Codes personalizados para links de produtos, facilitando o acesso rápido a páginas de vendas online.

🚀 Funcionalidades
Gerar QR Code a partir de links de produtos
Nomear automaticamente os arquivos gerados
Exportar QR Code em formato .png
Interface simples via terminal
🧠 Tecnologias utilizadas
Node.js
Biblioteca qrcode
Módulo nativo readline
⚙️ Como executar
Instale as dependências:
npm install qrcode
Execute o projeto:
node qr-generator.js
📦 Como funciona

O usuário informa:

Nome do produto
Link da página de venda

O sistema gera automaticamente um QR Code que pode ser usado para:

Divulgação de produtos
Marketing digital
Acesso rápido via celular
💡 Objetivo

Facilitar a geração de QR Codes para e-commerces de forma rápida, prática e sem necessidade de interface gráfica.

🔮 Melhorias futuras
Personalização de cores
Inserção de logo no QR Code
Geração em lote (vários produtos)
Integração com banco de dados
API para geração automática
